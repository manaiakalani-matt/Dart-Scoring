# Dart Checkout Flash Trainer

Fast orange-and-black flashcard app for learning dart checkouts with **per-dart remainder math**.

## Run

Open `index.html` in any browser.

## Features

- Checkout cards from 2 to 170 (excluding non-finish bogey numbers).
- Every dart shows:
  - hit target (`T20`, `D16`, `Bull`, etc.)
  - numerical value (`60`, `32`, `50`, etc.)
  - remaining score after the dart (e.g., `78 left`)
- Tap/click the card to flip between target and checkout route.
- Score tracking (`Knew it` / `Missed it`).
- Study ranges: all, pro zone, big finishes, doubles only.
- Keyboard controls:
  - `Space`: reveal/hide
  - `N`: next card
  - `1`: knew it
  - `2`: missed it
